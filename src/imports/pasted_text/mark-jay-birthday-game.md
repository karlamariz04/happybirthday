# 🎮 Figma AI Prompt – **The Grand Voyage of Mark Jay**

### A One Piece-Inspired Birthday Adventure Game (Android Mobile)

Create a **brand-new Android mobile adventure game** inspired by the adventurous spirit, colorful world, humor, and ocean exploration of **One Piece**. The game should feel like an anime RPG with pirate adventures, mysterious islands, treasure hunting, and heartfelt moments. It is a **birthday gift game** dedicated to **Mark Jay**.

> **Important:** Create **original characters inspired by the roles and personalities** of the Straw Hat Pirates. Do **not** copy official character designs, logos, or artwork. Capture the adventurous feeling while keeping the game visually original.

# Story

The Grand Ocean hides a legendary treasure that appears only once each year.

According to an ancient legend, only one worthy sailor can complete the voyage and discover the treasure.

This year's chosen adventurer is...

# **Mark Jay**

With the help of an adventurous pirate crew, Mark Jay must sail across dangerous seas, complete legendary challenges, collect four sacred relics, and unlock the treasure that awaits at the end of his journey.

Unknown to him, the greatest treasure is not gold...

It is a heartfelt birthday surprise.
